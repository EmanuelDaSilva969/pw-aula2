Evento: onMouseOut

Descrição

O evento onMouseOut é acionado quando o cursor do mouse sai de cima de um elemento HTML. É útil para criar interações que requerem a exibição ou ocultação de conteúdo quando o usuário passa o mouse sobre ou fora de um elemento.

Elementos envolvidos:

  • Qualquer elemento HTML pode ser alvo do evento onMouseOut.
  • É possível acessar o elemento que disparou o evento usando a propriedade "this" dentro da função de resposta.

Exemplo Funcional:

Codigo do exemplo Funcional

HTML:

<div class="myBox"></div>

JS:

document.querySelector('.myBox').addEventListener('mouseout', (e) => {
   alert('O mouse saiu na div');
});
