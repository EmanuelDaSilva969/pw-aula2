Evento: onClick

Descrição

O evento onclick é disparado quando o usuário clica em um elemento HTML. É uma forma comum de criar interações com botões, links e outros elementos clicáveis.

Elementos envolvidos:

  • Qualquer elemento HTML pode ser clicado para disparar o evento onclick.
  • É possível acessar o elemento que disparou o evento usando a propriedade "this" dentro da função de resposta.

Exemplo Funcional:

Codigo do exemplo Funcional

HTML:

<button id="myButton">Clique aqui</button>

JS:

document.querySelector('#myButton').addEventListener('click', () => {
   alert('O botão foi clicado!')
});
