Evento: onKeyUp

Descrição

Descrição: O evento onKeyUp é disparado quando uma tecla é solta após ter sido pressionada. Esse evento é comumente usado para implementar interações baseadas em teclado, como ativar uma ação quando o usuário pressiona uma determinada tecla. O evento é disparado no elemento que está com o foco na página, ou seja, aquele que está pronto para receber input do usuário.

Elementos envolvidos:

  • Qualquer elemento HTML pode ser usado para receber o evento onKeyUp, desde que tenha o foco na página. Para receber o evento, é necessário adicionar um listener para o evento onKeyUp no elemento desejado.
  • É possível acessar o elemento que disparou o evento usando a propriedade "this" dentro da função de resposta.

Exemplo Funcional:

Codigo do exemplo Funcional

HTML:

<input type="text"/>

JS:

document.querySelector('input').addEventListener('keyup', (e) => {
   alert(`A tecla ${e.key} foi solta`);
});
