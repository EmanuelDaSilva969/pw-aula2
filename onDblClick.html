Evento: dblClick

Descrição

O evento dblClick é disparado quando o usuário clica duas vezes em um elemento HTML em um curto período de tempo. É semelhante ao evento onClick, mas com a diferença de que ocorre com um duplo clique em vez de um único clique. Esse evento é útil para criar interações com elementos que requerem ações mais precisas e específicas, como selecionar um item em uma lista ou editar um campo de texto. Ao utilizar o evento dblClick, é possível criar experiências de usuário mais intuitivas e eficientes em sua aplicação ou site.

Elementos envolvidos:

  • Qualquer elemento HTML pode ser clicado para disparar o evento dblClick.
  • É possível acessar o elemento que disparou o evento usando a propriedade "this" dentro da função de resposta.

Exemplo Funcional:

Codigo do exemplo Funcional

HTML:

<button id="myButton">Clique aqui</button>

JS:

document.querySelector('#myButton').addEventListener('dblclick', () => {
   alert('O botão foi clicado duas vezes seguidas em um curto periodo de tempo!')
});
