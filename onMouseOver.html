Evento: onMouseOver

Descrição

O evento onMouseOver é disparado quando o ponteiro do mouse é posicionado sobre um elemento HTML. Ele é útil para criar interações com elementos que requerem ações adicionais quando o mouse está sobre eles, como exibir informações adicionais ou efeitos visuais.

Elementos envolvidos:

  • Qualquer elemento HTML pode ser alvo do evento onMouseOver.
  • É possível acessar o elemento que disparou o evento usando a propriedade "this" dentro da função de resposta.

Exemplo Funcional:

Codigo do exemplo Funcional

HTML:

<div class="myBox"></div>

JS:

document.querySelector('.myBox').addEventListener('mouseover', (e) => {
   alert('O mouse entrou na div');
});
