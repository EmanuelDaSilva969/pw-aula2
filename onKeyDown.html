Evento: onKeyDown

Descrição

O evento onKeyDown é acionado quando uma tecla é pressionada no teclado enquanto o foco está em um elemento HTML. Esse evento é útil para capturar entradas de teclado do usuário e executar ações com base nessas entradas. É semelhante ao evento onKeyPress, mas com a diferença de que também é acionado quando uma tecla não é suportada pela entrada de caracteres.

Elementos envolvidos:

  • Qualquer elemento HTML que possa receber o foco pode ser usado para acionar o evento onKeyDown. Normalmente, é usado em elementos como caixas de texto e áreas de texto, mas pode ser usado em qualquer elemento HTML com a propriedade tabIndex definida como true.
  • É possível acessar o elemento que disparou o evento usando a propriedade "this" dentro da função de resposta.

Exemplo Funcional:

Codigo do exemplo Funcional

HTML:

<input type="text"/>

JS:

document.querySelector('input').addEventListener('keydown', (e) => {
   alert(`A tecla ${e.key} foi pressionada`);
});
