Eventos em JavaScript

Descubra como utilizar os eventos em JavaScript para criar interatividade e melhorar a experiência do usuário em páginas web

O que são eventos em JavaScript?

Eventos em JavaScript são ações que ocorrem dentro de um documento HTML e que podem ser detectadas e respondidas por meio de código JavaScript. Eles permitem que você crie interações dinâmicas e responsivas com o usuário, como clicar em um botão, mover o mouse sobre um elemento ou digitar em um campo de texto.

Existe uma infinidade de eventos definidos para uso em JavaScript, porem nesta pesquisa trataremos especifcamente dos eventos abaixo:

Evento Descrição
onClick Ocorre quando o usuário clica em um elemento.
onDblClick Ocorre quando o usuário clica duas vezes em um elemento.
onChange Ocorre quando o valor de um elemento é alterado pelo usuário, geralmente usado em campos de formulário como caixas de seleção, botões de rádio e campos de texto.
onInput Ocorre quando o valor de um elemento é alterado pelo usuário ou programaticamente, semelhante ao evento onChange, mas dispara com mais frequência e funciona em outros elementos, como elementos div e span.
onLoad Ocorre quando um elemento, como uma imagem ou uma página, é completamente carregado.
onMouseOver Ocorre quando o ponteiro do mouse é movido sobre um elemento.
onMouseOut Ocorre quando o ponteiro do mouse é movido para fora de um elemento.
onKeyDown Ocorre quando uma tecla é pressionada enquanto o foco está em um elemento.
onKeyUp Ocorre quando uma tecla é liberada depois de ter sido pressionada enquanto o foco está em um elemento.
onFocus Ocorre quando um elemento recebe o foco, geralmente como resultado de um usuário clicando em um elemento ou navegando para ele usando a tecla Tab. É frequentemente usado para controlar a aparência do elemento quando ele recebe o foco.

Como sao representados?

Os eventos são representados no DOM (Modelo de Objeto de Documento) por meio de objetos Event, que contêm informações sobre o tipo de evento, o alvo do evento (ou seja, o elemento que disparou o evento) e outras informações relevantes. O DOM é uma representação em árvore dos elementos HTML em uma página, que pode ser manipulada por meio de código JavaScript.

Como lidar com eventos?

Para responder a um evento, é possível criar uma função em JavaScript que será chamada sempre que o evento for disparado. Existem duas maneiras principais de se fazer isso em JavaScript: usando a propriedade "on" do elemento HTML (como onclick, onmouseover, etc.) ou usando a função addEventListener.

A função addEventListener é considerada uma prática recomendada para lidar com eventos em JavaScript, pois permite adicionar vários ouvintes para um evento em um único elemento, garantindo que os ouvintes sejam chamados na ordem em que foram adicionados. Além disso, a função addEventListener também permite remover ouvintes para um evento específico, o que pode ser útil em determinadas situações.

A sintaxe básica da função addEventListener é a seguinte:

elemento.addEventListener(evento, função, useCapture);

Onde "elemento" é o elemento HTML ao qual você deseja adicionar um ouvinte de evento, "evento" é uma string que indica o tipo de evento que você deseja detectar (por exemplo, "click", "mouseover", etc.), "função" é a função JavaScript que será chamada sempre que o evento for disparado e "useCapture" é um valor booleano opcional que indica se o ouvinte deve ser chamado durante a fase de captura do evento (quando o evento é propagado para o elemento pai) ou durante a fase de bubbling (quando o evento é propagado para o elemento filho).

Exemplos

  • onClick
  • onDblClick
  • onChange
  • onInput
  • onLoad
  • onMouseOver
  • onMouseOut
  • onKeyDown
  • onKeyUp
  • onFocus

Conclusão

Em resumo, eventos em JavaScript são ações que ocorrem dentro de um documento HTML e que podem ser detectadas e respondidas por meio de código JavaScript. Eles são representados no DOM por meio de objetos Event e podem ser tratados usando a propriedade "on" do elemento HTML ou a função addEventListener, que é considerada uma prática recomendada para lidar com eventos em JavaScript.
