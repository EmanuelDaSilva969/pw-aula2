Evento: onFocus

Descrição

Descrição: O evento onFocus é acionado quando um elemento HTML recebe o foco, ou seja, quando o usuário seleciona ou ativa o elemento para interagir com ele. Esse evento é útil para realizar ações específicas quando um elemento recebe o foco, como alterar a cor de fundo ou exibir informações adicionais.

Elementos envolvidos:

  • Qualquer elemento HTML pode receber o foco e acionar o evento onFocus. É possível utilizar o evento em elementos como inputs, botões e links para melhorar a experiência do usuário ao navegar pelo site ou aplicação.
  • É possível acessar o elemento que disparou o evento usando a propriedade "this" dentro da função de resposta.

Exemplo Funcional:

Codigo do exemplo Funcional

HTML:

<input type="text" placeholder="Digite seu email" autocomplete="none"/>

<p id="demo"></p>

JS:

document.querySelector('input').addEventListener('focus', (e) => {
   document.querySelector('#demo').textContent = "O input recebeu o foco";
});
document.querySelector('input').addEventListener('blur', (e) => {
   document.querySelector('#demo').textContent = "O input perdeu o foco";
});
